Quick Start

Get started with LucenDEX API integration in 3 steps:

  1. Request Quote: POST to /v1/quote with pair + amount
  2. Sign Transaction: Embed QuoteHash in memo, sign client-side
  3. Submit: POST to /v1/submit or direct to rippled

Architecture Overview

LucenDEX is built on four core principles:

  • Non-Custodial: Backend never sees private keys
  • Deterministic: QuoteHash binding ensures verifiable execution
  • Zero-Trust: mTLS between services, Ed25519 auth for partners
  • Data Minimization: DEX-only data, no PII, no user accounts
